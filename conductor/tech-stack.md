# Tech Stack - Mika Lending Bot

## Backend
- **Language:** Python 3.12+
- **Numerical Processing:** NumPy
- **Data Analysis:** Pandas
- **HTTP Client:** Requests
- **Timezone Management:** Python Native (datetime.UTC)

## Development & Tooling
- **Package Manager:** uv
- **Linter & Formatter:** Ruff
- **Type Checker:** MyPy (Strict Mode)
- **Task Runner:** Poe the Poet
- **Build System:** Hatchling
- **Database:** SQLite (for local market data and lending history)

## Testing
- **Framework:** Pytest
- **Property-Based Testing:** Hypothesis
- **Coverage:** Pytest-Cov

## Frontend & Web
- **Interface:** Vanilla HTML/JavaScript/CSS (Legacy Web UI)
- **Server:** Python-based Web Server (internal module)

## Infrastructure
- **Containerization:** Docker
- **Orchestration:** Docker Compose
